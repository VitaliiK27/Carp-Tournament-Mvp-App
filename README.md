# Carp Tournament MVP

MVP-додаток для обліку результатів карпових змагань.

## Функції
- додавання команд
- додавання зважувань
- топ-5 риб по команді
- загальна вага
- переможці зон
- Big Fish
- подіум

## Локальний запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Windows
```
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
## Як запустити локально
У папці проєкту:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
